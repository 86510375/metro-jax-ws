Checking out source code

If you are new to java.net, check out this page for how to access the CVS repository. Access to the sources are under the dual license consisting of the CDDL v1.0 and GPL v2.

The following map shows the trail map of major branches in the repository. 

    |-------->fcs-20
    |
==============================================> trunk
       |                |
       |------>mr-21    |----->jaxws212
  • The current development is taking place in trunk and carries the future JAX-WS RI version.
  • jaxws212 : JAX-WS RI 2.1.2 branch.
  • mr-21 : JAX-WS RI 2.1.1 branch.
  • fcs-20 : JAX-WS RI 2.0 branch.

Getting Started

After checking out the repository you can build the workspace and run the samples by following the steps. 

$ ant

Among other things this run code generators. Once you build a workspace, you can then use your favorite IDE to hack the code.
